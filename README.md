# ElVino

ElVino er en maskinlærings algoritme som forteller deg om en vin er av høy eller lav kvalitet. Dette gjør den basert på den kjemiske oppbygningen til vinen. Modellen er trent på UC Irvine Machine Learning Repository sitt [Wine](https://archive.ics.uci.edu/dataset/109/wine) datasett

Nettsiden er hostet via Gradio: https://2a1aaff4286ee1f495.gradio.live (Link generert 02.11.25)
